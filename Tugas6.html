    File css dan html bisa digabung ataupun dipisah
    CSS sangat berguna untuk user interface
    Fungsi tag div hampir sama dengan tag paragraf, berguna untuk mengelompokkan sejumlah baris teks yang memiliki karakteristik yang sama.
    Display : block berfungsi untuk dapat membuat tombol tabel seperti layaknya blok blok.
    a:link Keadaan awal pemicu link aktif
    a:visited Keadaan pemicu link setelah dikunjungi
    a:active Keadaan pemicu yang sedang aktif
    a:hover Kejadian pada pemicu link saat mouse digerakkan diatasnya.
    Href berfungsi untuk memberikan hyperlink pada suatu teks.
    Border-style : solid berfungsi untuk mengatur style border, solid artinya terlihat dengan jelas. Border color : blue berfungsi untuk mengatur warna border
    Font-family berfungsi untuk memilih font yang akan digunakan pada website kita. Font-size berfungsi untuk mengatur ukuran font itu sendiri. Color berfungsi untuk mengatur warna font. Background color berfungsi untuk mengatur warna background web kita.
    Padding adalah sejenis margin, tetapi padding menggunakan pengaturan halaman dengan menggunakan satuan persentase. Sama seperti margin, ia memiliki 4 macam arah yaitu top, bottom, left, right.
